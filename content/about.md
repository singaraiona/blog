Hi, I'm Anton Kundenko, a seasoned engineer from Ukraine with a passion for developing programming languages and database engines. Over the years, I've honed my skills and now I'm leading the charge on an exciting project called [RayforceDB](https://rayforcedb.com/), a vector database built on the interpreter of the Lisp-like language [Rayfall](https://rayforcedb.com/content/syntax.html). This journey is filled with fascinating challenges that require innovative solutions, which inspired me to create this blog. I hope you'll find something intriguing and insightful here.
